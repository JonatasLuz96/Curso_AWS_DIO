## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Jonatas Luz. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especí­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- Amazon S3 (Simple Storage Service)
- Armazenamento de arquivos e dados na nuvem com baixo custo.
- Armazenar documentos regulatórios, relatórios, notas fiscais e backups, eliminando servidores físicos de arquivos.

Etapa 2: 
- Amazon EC2 (Elastic Compute Cloud)
- Hospedagem de servidores virtuais sob demanda.
- Executar sistemas internos da empresa, como controle de estoque e gestão administrativa, pagando apenas pelo tempo de uso.

Etapa 3: 
- Amazon RDS (Relational Database Service)
- Banco de dados relacional gerenciado pela AWS.
- Armazenar dados de produção, medicamentos, clientes e fornecedores, com backups e segurança automáticos.



## Conclusão
A empresa Abstergo Industries, ao adotar os serviços Amazon S3, EC2 e RDS, consegue reduzir custos operacionais, eliminar gastos com infraestrutura física, aumentar a segurança dos dados e escalar seus sistemas conforme a necessidade.
Essa solução em nuvem permite maior eficiência operacional, mantendo o foco no crescimento e inovação da empresa.
